# indy-louver-designer

Important Considerations for Building This ProjectIf you move forward with building this structure, the interactive layout reveals a few critical design constraints you wouldn't see on paper:The Shadow Shift: Because your slats are 10 feet in the air, the shadow they cast will be shifted horizontally away from the structure. In the summer, a $73.7^\circ$ sun angle means the shadow shifts forward by about 3 feet ($10 \div \tan(73.7^\circ)$). In the winter, the low $26.8^\circ$ sun shifts the sunlight patch nearly 20 feet north of the structure overhead.The Louver Orientation: This mathematical sweet spot only works if the slats run precisely East to West. If the patio layout forces the slats to run North to South, fixed slats cannot optimize for seasons this way; you would need a mechanical, adjustable louver system instead to track the sun from east to west throughout the day.
